== Bugfixes

=== https://issues.apache.org/jira/browse/CXF-9214[CXF-9214] WSDLs and XSDs cannot be loaded from class path on GraalVM

A workaround was added for https://issues.apache.org/jira/browse/CXF-9214[CXF-9214],
where WSDLs and XSDs could not be loaded from the class path when running on GraalVM native image.
See also the upstream https://github.com/apache/cxf/pull/3124[pull request].
